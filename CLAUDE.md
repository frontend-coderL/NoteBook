# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 仓库性质

个人 Mac 装机配置知识库，内容全为 Markdown 笔记，无构建系统、无测试框架、无依赖管理文件。

## 内容结构

```
Mac 装机/
├── Claude Code.md      # Claude Code 安装、插件、Skills 配置
├── Mac 工具软件.md      # 工具软件推荐（启动器、截图、播放器等）
├── Mac 效率配置.md      # 系统效率配置（键盘、触控板、Dock 等）
├── Mac 终端配置.md      # 终端环境配置（Homebrew、Ghostty、Oh My Zsh、Starship）
├── Node 版本管理.md     # Node.js 版本管理工具（待填充）
└── uv 管理 python.md   # Python uv 工具配置（待填充）
```

## 编写约定

- 工具清单中 ✨ 标记表示该类别的首选推荐项
- 新增工具遵循现有格式：`### 编号 工具名✨`，包含官网链接、特点说明、安装命令三段
- 代码块语言标注统一使用 `Shell`（与现有文档一致，非 `bash`）
- 笔记用中文书写，命令保持英文
