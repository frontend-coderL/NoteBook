## mise 管理版本

> [https://github.com/jdx/mise](https://github.com/jdx/mise)

### 安装 mise

```Shell
# 使用homebrew安装mise
brew install mise

# 初始化 shell：eval "$(mise activate zsh)"
echo 'eval "$(mise activate zsh)"' >> ~/.zshrc
source ~/.zshrc
```

### 管理 Node

```Shell
# 安装 node@lts，并将其设为全局默认
mise use --global node@lts

# 进入项目设置 node 版本
# 会生成 mise.toml
mise use node@22
```

### 管理 java

```Shell
mise use java@21
mise use java@17 --global
```



## corepack 管理 npm/pnpm

> Corepack 是 Node.js 官方内置的一层“包管理器代理（package manager shim）”，用于统一管理 pnpm / yarn（以及未来扩展） 的版本。

```Shell
# 启用 corepack
corepack enable

# 设置 pnpm（全局默认）
corepack prepare pnpm@latest --activate
```

```JSON
{
  "packageManager": "pnpm@10.33.2"
}
```

## uv 管理 Python

> [GitHub - astral-sh/uv: An extremely fast Python package and project manager, written in Rust.](https://github.com/astral-sh/uv)

安装 uv

```Shell
curl -LsSf https://astral.sh/uv/install.sh | sh
# 或者
brew install uv

uv --version
```

安装 Python

```Shell
# 查看可用版本：
uv python list

# 安装一个稳定版本（建议 LTS）：
uv python install 3.12
```

全局配置私有源

```Shell
echo 'export UV_INDEX_URL=https://pypi.company.com/simple' >> ~/.zshrc
echo 'export UV_EXTRA_INDEX_URL=https://pypi.org/simple' >> ~/.zshrc

source ~/.zshrc
```

项目配置私有源

```Shell
# 腾讯云源

# url = "https://mirrors.tencentyun.com/pypi/simple/"  # 内网
```

### 现代 Python 工作流

```Shell
# 初始化项目
uv init my-project
# 创建 Python 环境
uv venv --python 3.12

# 添加依赖
uv add fastapi uvicorn
# 从 requirements 导入依赖
uv add -r requirements.txt

# 运行项目
uv run python main.py
```

### 传统 venv 模式（兼容）

```Shell
# 创建虚拟环境
uv venv --python 3.12

# 安装依赖
uv pip install fastapi uvicorn
# 或 requirements：
uv pip install -r requirements.txt

uv run python main.py
```
