## **系统效率配置**

### **键盘**

- **F1-F12 作为标准功能键**
  - 路径：系统设置 → 键盘 → 键盘快捷键 → 功能键

  - 开启：✔️ 勾选「将 F1、F2 等键用作标准功能键」

- **按键重复速度优化**
  - 路径：系统设置 → 键盘

  - 建议调整：
    - 按键重复速度：调到「最快」

    - 重复前延迟：调到「最短」

### **触控板与鼠标**

- **轻点来点按**
  - 路径：系统设置 → 触控板

  - 开启：✔️ 轻点来点按

- **辅助点按（右键）**
  - 路径：系统设置 → 触控板

  - 开启：✔️ 辅助点按

  - 选择：点按右下角

- **三指拖移**
  - 路径：系统设置 → 辅助功能 → 指针控制 → 触控板选项

  - 开启：✔️ 启用“三指拖移”

- **四指手势**
  - 路径：系统设置 → 触控板 → 更多手势

  - 建议：全部改成四指触发，避开三指，防止手势冲突。
    - 四指上滑：调出调度中心，所有窗口、桌面一眼看完

    - 四指下滑：APP 预览，只看当前软件的所有窗口

    - 四指左右滑：快速切换全屏应用、不同桌面空间

    - 四指张开：一键显示桌面，临时藏起所有窗口

    - 四指捏合：打开软件搜索

- **滚动方向**
  - 路径：系统设置 → 触控板 / 鼠标

  - 建议：
    - 如果偏 Windows：关闭“自然滚动”

    - 如果偏 iOS：保持开启

### **Finder**

- **显示路径栏与状态栏**
  - 路径：Finder → 显示

  - 开启：
    - ✔️ 显示路径栏

    - ✔️ 显示状态栏

- **默认以列表视图打开**
  - 路径：Finder → 显示方式

  - 建议：默认列表视图（List View）

## **通用效率**

### **启动器与全局搜索**

- [raycast](https://raycast.com)：现代化 macOS 启动器，支持应用启动、系统控制、脚本和插件扩展，适合构建个人效率工作流。
  - 特点：适合开发者和重度效率用户，工作流自动化能力强。

- [utools](https://www.u.tools) ⭐：跨平台插件化工具箱，适合快速调用翻译、OCR、JSON 等小工具。
  - 特点：适合轻量工具需求用户，开箱即用。

### **剪贴板**

- [Paste](https://pasteapp.io)：macOS 剪贴板管理工具，支持多类型历史记录（文本、图片、文件等），可快速检索和回溯复制内容。
  - 特点：界面精致、体验流畅，适合重度复制粘贴用户。

- [PasteNow](https://pastenow.app) ⭐：剪贴板历史工具，支持多设备同步与快捷检索，偏国内用户使用场景优化。
  - 特点：功能实用、轻量，适合基础剪贴板增强需求。

- [iCopy](https://www.aodo365.com/icopy.html)：轻量剪贴板增强工具，提供基础复制记录与管理能力。
  - 特点：简单易用，适合只需要基础历史记录的用户。

- [PasteMemo](https://www.lifedever.com/PasteMemo/)：轻量级原生 macOS 剪贴板管理器，常驻菜单栏，自动记录复制内容并支持快速检索。
  - 仓库：[PasteMemo-app](https://github.com/lifedever/PasteMemo-app)。

### **截图与录屏**

- [Snipaste](https://www.snipaste.com) ⭐：强大的截图与贴图工具，支持截图后快速标注并“贴在桌面上”长期查看。
  - 特点：贴图能力极强，适合设计、开发与信息对照场景。

- [Pinpix](https://pinpix.app)：轻量截图与标注工具，主打快速截取与简单编辑。
  - 特点：轻量快速，适合基础截图标注需求。

- [Snippai](https://github.com/xyTom/snippai)：AI 驱动的智能截图工具，支持从截图中提取和理解信息。
  - 功能亮点：公式识别、文字提取、表格转换、图像分析、问题求解、代码理解、色彩提取、语言翻译。

- [macshot](https://github.com/sw33tLie/macshot)：原生 macOS 截图和录制工具，支持标注、自动遮蔽隐私、GIF 录制、OCR、滚动截图等能力。
  - 特点：基于 Swift + AppKit 开发，轻量且功能全面。

### **窗口管理**

- [Magnet](https://magnet.crowdcafe.com) ⭐：窗口分屏与管理工具，支持快捷键快速将窗口排列到屏幕不同区域。
  - 特点：高效窗口布局工具，适合多任务办公场景。

## **系统增强**

### **菜单栏管理**

- [Thaw](https://github.com/stonerl/Thaw) ⭐：菜单栏管理工具，用于隐藏和显示菜单栏项目。

- [Bartender](https://www.macbartender.com)：macOS 菜单栏管理工具，可隐藏、整理和按规则显示菜单栏图标。
  - 特点：适合菜单栏图标较多、需要长期整理的重度用户。

- [iBar](https://www.better365.cn/ibar.html)：轻量菜单栏图标隐藏工具，用于收纳不常用状态栏图标。
  - 特点：适合不想付费、只需要基础隐藏功能的用户。

- [Ice](https://github.com/jordanbaird/Ice)：用于隐藏并整理菜单栏图标的管理工具。

- [Itsycal](https://github.com/sfsam/Itsycal/tree/master) ⭐：状态栏日历与时间小组件，适合替换系统默认时间显示。

### **显示与桌面**

- [BetterDisplay](https://github.com/waydabber/BetterDisplay)：显示器增强工具，可调节亮度、分辨率和显示模式。

- [DockDoor](https://github.com/ejbills/DockDoor)：为 macOS 提供窗口预览、`Alt-Tab` 风格切换等增强能力。

- [SuperIsland](https://github.com/shobhit99/SuperIsland)：将 MacBook 刘海区域改造成可展示信息和交互的小面板。

- [Cadran](https://cadranapp.com/)：在桌面壁纸和屏幕保护程序上显示多种可自定义时钟表盘。

### **鼠标与输入增强**

- [Mos](https://mos.caldis.me) ⭐：鼠标滚动增强工具，优化外接鼠标滚动体验，使其更平滑自然。
  - 特点：解决 Mac 鼠标滚动不顺滑问题，提升外接鼠标体验。

### **压缩与解压**

- [Keka](https://www.keka.io) ⭐：macOS 轻量级压缩解压工具，支持多种格式（7z、zip、rar 等），可拖拽快速处理文件压缩与解压。
  - 特点：免费轻量、右键即用，适合日常文件压缩解压需求。

- [ShichiZip](https://github.com/idawnlight/ShichiZip)：复现 7-Zip File Manager 的完整体验，用 macOS 原生方式重做。

### **包管理辅助**

- [WailBrew](https://github.com/wickenico/WailBrew)：Homebrew 可视化工具，适合查看和管理已安装的软件包。

### **系统清理**

- [Mole](https://github.com/tw93/mole) ⭐：深度清洁和优化你的 Mac。
  - `mo`：交互式菜单。

  - `mo clean`：深度清理，`mo clean --dry-run` 用于预览清理内容。

  - `mo uninstall`：智能卸载应用。

  - `mo optimize`：系统优化。

  - `mo purge`：项目构建产物清理。

  - `mo installer`：清理安装包。

## **开发常用**

### **AI 开发工具**

- [codux](https://github.com/duxweb/codux) ⭐：原生 macOS 终端工作区，专为 AI 编程工具打造，支持多项目管理、内置 Git 和 AI 用量追踪。

### **IDE 与编辑器**

- [CodeKey Run](https://ckey.run/)：一行命令激活 JetBrains 全家桶，适合快速打开项目和工具链。

- [VS Code](https://github.com/microsoft/vscode)：主流跨平台代码编辑器，插件生态完善，适合作为默认开发编辑器。

### **数据库与 Redis 客户端**

- [openhare](https://github.com/sjjian/openhare)：AI 驱动的桌面 SQL 客户端，跨平台，使用 Flutter 构建。

- [RedisME](https://www.hepengju.com/zh/)：Redis 图形化客户端，适合日常查看、编辑和管理 Redis 数据。

- [TinyRDM](https://github.com/tiny-craft/tiny-rdm)：轻量 Redis 桌面客户端，适合本地和远程 Redis 连接管理。

### **Markdown 写作**

- [Typora](http://www.typora.io/) ⭐：基于 Electron 的“读写一体” Markdown 编辑器。

- [Ferrite](https://github.com/OlaProeis/Ferrite)：轻量原生文本编辑器，支持 Markdown、JSON、YAML 和 TOML。

- [note-gen](https://github.com/codexu/note-gen)：专注记录和写作的跨端 AI 笔记工具。

- [mdit](https://github.com/hjinco/mdit)：现代本地优先的 Markdown 笔记应用。
  - 普通 `.md` 文件留在本地，数据可控。

  - 提供接近 Notion 的编辑体验，适合思考和组织内容。

  - 支持类似 Obsidian 的反向链接和图谱视图。

  - 支持隐私优先的 AI 能力，可接入自己的 API Key 或本地模型。

- [open-notebook](https://github.com/lfnovo/open-notebook)：NotebookLM 的开源替代实现，强调更高的灵活性和扩展空间。

- [BiliNote](https://github.com/JefferyHcool/BiliNote)：AI 视频笔记生成工具，适合把视频内容整理成结构化笔记。

### **API 调试与抓包**

- [Reqable](https://reqable.com/zh-CN/) ⭐：集抓包、API 调试、Mock 和测试于一体的国产解决方案，适合替代多个独立工具。

- [Postman](https://www.getpostman.com/)：常用 API 调试工具，适合接口调试和团队协作。

- [Apifox](https://www.apifox.cn/)：集 API 文档、调试、Mock、自动化测试于一体的协作平台。

- [ReqRes](https://reqresapp.com/)：用于监视、调试和模拟 HTTP(S) 请求与响应的原生 macOS 应用。

- [Charles](https://www.charlesproxy.com/)：经典代理抓包工具，可查看和分析 HTTP / HTTPS 流量。

- [Proxyman](https://proxyman.app/) ⭐：现代化 macOS HTTP 调试代理，界面更直观，使用体验更友好。

- [Wireshark](https://www.wireshark.org/)：最常用的网络协议分析工具，适合更底层的网络排查。
